# Skill 工作流

这里是我用来沉淀**可复用 Skill / Agent 工作流**的地方。

## 这个文件夹放什么

- 我自己写的 Claude / Agent **skill**（包含 SKILL.md、prompt、reference 资料、示例）
- 把多步任务编排成的**工作流**（例如：从 PRD → 拆任务 → 实现 → 自检 → 提交）
- 工作流落地中的**复盘和坑表**（哪些做法被验证、哪些只是看上去好用）

## 这个文件夹不放什么

- 一次性、不可复用的脚本和片段（这些更适合放在 `瞎鼓捣工具/`）
- 只是想法的草稿（草稿先写在 `notes/`）

## 索引

每写一个新的 Skill 工作流，请在这里追加一行链接，方便从首页直接跳转。
