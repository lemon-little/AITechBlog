## Codebase Patterns
- Hugo version is 0.154.5 (extended). The deprecated top-level `paginate` key must be written as `[pagination]\n  pagerSize = N` instead, otherwise build fails.
- Theme is `hugo-theme-cleanwhite` and lives as a git submodule under `themes/hugo-theme-cleanwhite`. It uses `params.addtional_menus` (note the typo "addtional") for the top navigation.
- Content lives under `content/post/<中文目录>/`. The cleanwhite theme picks up everything under `content/post/` automatically; an `_index.md` is what makes a sub-folder a real Hugo section page (NOT `README.md`).
- The site is deployed to GitHub Pages via `.github/workflows/` on push to `main` — production baseURL is `https://lemon-little.github.io/AITechBlog/`.
